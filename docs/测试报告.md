## 3. 测试过程中发现的问题与处理

### 3.1 未登录接口状态码不准确

首次执行自动化脚本时，认证模块有 1 个用例失败：

| 用例 | 脚本预期 | 实际返回 |
| --- | --- | --- |
| 无 Token 访问 `/api/auth/me` | HTTP `401` | HTTP `403` |
| 无效 Token 访问 `/api/auth/me` | HTTP `401` | HTTP `403` |

处理方式：

- 在后端 `SecurityConfig` 中补充 `authenticationEntryPoint`。
- 未登录或登录失效统一返回：

```json
{"code":401,"message":"未登录或登录已失效","data":null}
```

修复后重新打包后端并复测，认证模块全部通过。
